Command Cheatsheet

Type these commands inside your text. The app automatically converts single brackets [] to the required double brackets [[]].

Pauses & Silence

  • [500] : Shortcut for 500ms silence.
  • [slnc 500] : Explicit command for 500ms silence.

Emphasis & Stress

  • [emph] : Emphasize the NEXT word.

Voice Properties

  • [rate 200] : Change speed to 200 WPM.
  • [vol 0.5] : Set volume to 50%.
  • [pitch 80] : Set pitch to 80 (range 0-99).

Phonetic Input

You can write raw phonemes by enclosing them in brackets:

  • [DIs Iz f@n'EtIk] : "This is phonetic"
